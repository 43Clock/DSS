Use Case: Sistema comunica ordem de transporte
Pré-Condição: Existe uma palete em estado de espera
Pós-Condição: O robot recebe a instrução
Fluxo Normal:
    1. Sistema localiza as paletes
    2. Sistema procura as prateleiras onde estão armazenadas as paletes
    3. Sistema atualiza o nivel de ocupacao da zona de armazenamento
    4. Sistema atualiza o nivel de ocupação das prateleiras
    5. Sistema procura robots para transportar as paletes
    6. Sistema calcula os percursos para os robots
    7. Sistema cria as instruções
    8. Sistema envia as instruções aos robots
Fluxo Alternativo 1[palete encontra-se na zona de entrada](passo 1):
    1.1. Sistema procura prateleiras para colocar palete
    1.2. Sistema atualiza o nivel de ocupacao da zona de armazenamento
    1.3. Sistema atualiza o nivel de ocupação da prateleira
    1.4. Regressa a 5
Fluxo Alternativo 2:[não existem robots disponiveis](passo 5)
    5.1 Sistema aguarda até existir robot disponivel
    5.2 Regressa a 6

    1. Leitor de QrCode lê codigo da palete
    2. Leitor de QrCode comunica informação da leitura ao Sistema

    4. <<Include>> Sistema comunica ordem de transporte
